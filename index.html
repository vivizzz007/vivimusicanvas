Contribute on GitHub

How to add a Canvas

01

Upload Video

Place your .mp4 file in the Song/ or Album/ folder.

Tip: Keep file sizes small (< 5MB) for instant loading.

02

Update JSON

Add a new entry to canvas.json mapping the Song Title and Artist to your public URL.

03

Enable in App

In ViviMusic settings, enable "Show motion artwork". Play your song & enjoy the visuals!

Technical Requirements

  • Format: .mp4 files only.
  • Aspect Ratio: 9:16 (Vertical) is highly recommended.
  • File Size: Maximum 5MB (smaller is better).
  • Looping: 5-20 seconds for seamless transition.
